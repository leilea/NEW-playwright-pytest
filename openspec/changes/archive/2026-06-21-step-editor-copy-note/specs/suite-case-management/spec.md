## MODIFIED Requirements

### 需求:步骤编辑区

步骤编辑区必须提供完整的步骤增删改查功能，支持复制步骤和添加备注。

#### 场景:复制步骤
- **当** 用户点击步骤操作列中的复制按钮
- **那么** 系统深拷贝当前步骤，在步骤正下方插入新步骤，所有步骤重新编号

#### 场景:添加备注
- **当** 用户在备注列文本框中输入文字
- **那么** 该步骤的 note 属性同步更新为输入内容

#### 场景:删除步骤
- **当** 用户点击操作列的删除按钮
- **那么** 系统删除当前步骤，其余步骤重新编号

#### 场景:调整步骤顺序
- **当** 用户点击上移/下移按钮
- **那么** 当前步骤与相邻步骤交换位置，所有步骤重新编号
