## 新增需求

### 需求:回放脚本每步前植入步骤标记
生成的回放 pytest 脚本必须在每个交互步骤的代码前打印一个步骤标记，用于在失败时精确定位到对应的步骤序号。

#### 场景:单步步骤标记
- **当** 回放脚本包含一个步骤（如 click、fill、hover）
- **那么** 生成代码中在该步骤的 Playwright 调用前插入 `print('__STEP_MARKER__:{i}')`，其中 i 为 0 起始的步骤索引

#### 场景:导航型 click 标记
- **当** click 步骤触发了页面导航（`expect_navigation`）
- **那么** 步骤标记必须在 `with expect_navigation` 代码块之前打印
- **那么** 标记位置确保在导航超时失败时仍能识别到对应的步骤

#### 场景:非交互步骤不标记
- **当** 步骤是 wait、screenshot、scroll、eval 等非交互动作
- **那么** 仍然需要步骤标记，保持所有步骤都有对应的标记
