## ADDED Requirements

### 需求:定位器修改保存一致性

系统必须确保用户对步骤定位器的修改在保存后正确持久化。

#### 场景:编辑定位器后直接保存
- **当** 用户在步骤编辑器中修改某步骤的定位器输入
- **且** 用户点击保存按钮
- **那么** 该步骤的定位器值必须被持久化到存储
- **那么** 保存后页面必须显示最新定位器值

#### 场景:编辑定位器后修改同步骤其他字段再保存
- **当** 用户修改某步骤的定位器
- **且** 用户随后修改同步骤的其他字段（如 action、参数值等）
- **且** 用户点击保存按钮
- **那么** 已修改的定位器值必须被保留
- **那么** 保存后页面必须显示最新定位器值
